1. Назначение проекта
Разработка Telegram-бота, имитирующего работу банковской системы с базовыми функциями:
✅ Регистрация и авторизация пользователей
✅ Управление балансом
✅ Переводы между пользователями
✅ История операций

2. Требования к функционалу
Функция	Описание
Регистрация	Ввод ФИО, номера телефона, пароля. Данные сохраняются в Excel.
Авторизация	Проверка номера и пароля. Доступ к аккаунту.
Пополнение баланса	Кнопка "+1000 руб." для тестового пополнения.
Переводы	Выбор получателя и суммы. Проверка достаточности средств.
История операций	Просмотр всех переводов (от кого/кому, сумма).
Выход из аккаунта	Возврат в стартовое меню.

3. Технические требования
Компонент	Описание
Язык	Python 3.9+
Библиотеки	telebot, pandas
Хранение данных	Excel-файлы (База_данных.xlsx, База_переводов.xlsx)
Интерфейс	Кнопки Telegram (ReplyKeyboardMarkup)

4. Этапы разработки
Проектирование
Создание структуры базы данных.
Проработка сценариев взаимодействия.
Реализация
Написание кода (регистрация, авторизация, переводы).
Тестирование функций.
Документирование
Написание ТЗ, пояснительной записки.
Подготовка презентации.

5. Критерии приемки
✔ Бот корректно регистрирует новых пользователей.
✔ Переводы проходят без ошибок (проверка баланса, получателя).
✔ Данные сохраняются в Excel и не теряются при перезапуске.
✔ История операций отображается без искажений.
